# react-github-battle 
> Created as part of ReactTraining Course

Click here it see [Live demo](https://github-battle-cfea2.firebaseapp.com)
## Getting Started
### Setup
```
$ git clone https://github.com/react-github-battle.git
$ cd react-github-battle
```
### Install
```
$ npm i
```
### Run
``` 
$ npm start
```
